Um pouco sobre o site

Este é o meu primeiro site, desenvolvido utilizando apenas HTML e CSS. A ideia foi criar uma estrutura simples, mas funcional, para aprender os conceitos fundamentais do desenvolvimento web. O HTML foi utilizado para organizar o conteúdo e dar a base estrutural ao site, enquanto o CSS foi aplicado para melhorar a aparência, tornando o visual um pouco mais agradável. Como é o meu primeiro projeto, o foco foi garantir que o site fosse fácil de navegar, acessível e direto, sem complicações. As páginas foram pensadas para apresentar de forma clara os conceitos de HTML e CSS, proporcionando uma boa introdução a essas linguagens. A navegação é simples e intuitiva, e o design foi pensado para ser leve, proporcionando uma experiência de navegação agradável enquanto aprendo e exploro o desenvolvimento web. .
